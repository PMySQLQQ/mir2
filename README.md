## The Legend of Mir 2
《热血传奇2》是一个基于精灵的等距三维大型多人在线角色扮演游戏。玩家可以选择不同的职业，道士可以使用魔法召唤宠物协助战斗，毒害敌人并治愈盟友；战士是所有职业中防御和攻击属性最高的；巫师依靠魔法，可以轻松地使用法术杀死多个怪物；刺客则成为主要的近战伤害职业，但其防御属性比战士弱得多；弓箭手则是一个极其准确和有力的职业，使用强大的弓技能从远处造成特殊的伤害。
## Crystal Source
《热血传奇2》水晶源码是一个公共仓库，托管在 GitHub 上，旨在开发用 C＃编写的服务器和游戏客户端。

## Installation
1. 安装带有.NET包的Microsoft Visual Studio。
2. 下载最新的源代码（https://github.com/Suprcode/mir2/releases）。
3. 打开项目并配置 NuGet 包。
